Os formulários são definidos pela tag "form". Action é para onde os dados do form serão enviados. method é o jeito que eles serão enviados

O "for" na label corresponde a "id" do input. Nesse caso, o for="nome" e id="nome".

A tag "type" define o tipo do campo. Nesse caso "type=text" ou seja, igual a texto

Alguns dos tipos de input, são: button, checkbox, color, date, datetime, email, file, text, number e url

POST – é usado quando o cliente deseja enviar dados para processamento ao servidor, como os dados de um formulário, por exemplo.
